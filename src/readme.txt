It provides an annotation feature that is compatible with WP-Note.

== Description ==

The main function is as follows.

* Annotation functionality that is compatible with WP-Note ( http://wordpress.org/plugins/wp-note/ )
* Design that does not depend on the resolution ( Flexible & Flat )

= How to use =

1. Show me the post page
1. Write annotation text with shortcode

= Shortcode =

Shortcode is as follows.

    [note]ANNOTATION TEXT[/note]

    [important]ANNOTATION TEXT[/important]

    [tip]ANNOTATION TEXT[/tip]

    [warning]ANNOTATION TEXT[/warning]

    [help]ANNOTATION TEXT[/help]

Is compatible with WP-Note short code.

= Links =

* *Repository*: https://github.com/akabekobeko/wp-note-flexible2

== Installation ==

1. Upload 'wp-note-flexible2' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
